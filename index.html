Cette page est un outil de comptabilisation de réponses de questionnaires de satisfaction.

Mode opératoire

1- Sélectionnez le questionnaire souhaité

2- Renseignez le nom du fichier souhaité

3- Saisissez toutes les fiches. Le passage à une nouvelle fiche se fait en cliquant sur "Saisir la prochaine fiche".

4- Terminez en cliquant "Générer un fichier Excel"
